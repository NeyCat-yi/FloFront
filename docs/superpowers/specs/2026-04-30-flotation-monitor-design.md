# 铅锌矿浮选智能检测与决策系统 — 前端设计文档

## 概述

铅锌矿浮选过程的智能检测与决策系统前端页面，使用模拟数据搭建，用于展示浮选工艺的实时监控、参数调节、报警管理和数据分析功能。

## 技术栈

- React 18 + Vite（构建工具）
- Ant Design 5（深色主题）
- ECharts + echarts-for-react（图表与流程图）
- Mock 数据（前端内置）

## 页面布局

采用 CSS Grid 三列布局：

```
┌──────────────────────────────────────────┐
│              Navbar（全宽）                │
├───────────────────────────┬───────────────┤
│                           │  AlarmPanel   │
│       FlowChart           │               │
│   (突出显示，横跨左+中上)   ├───────────────┤
│                           │               │
├─────────────┬─────────────┤  ChartPanel   │
│  Realtime   │ ParamTable  │  (多图表)      │
│  Monitor    │  (可编辑)    │               │
└─────────────┴─────────────┴───────────────┘
```

- 左列约 30%，中列约 35%，右列约 35%
- FlowChart 横跨左列+中列上半部分，突出显示
- 下方左中并列：RealtimeMonitor 和 ParamTable
- 右侧上半 AlarmPanel，下半 ChartPanel
- 整体深色工业风：深蓝/深灰底色，青色/橙色高亮

## 组件设计

### Navbar（顶部菜单栏）

- 深色背景，左侧显示系统名称"铅锌矿浮选智能检测与决策系统"
- 四个下拉菜单：智能检测、优化决策、系统模式、用户管理（每个含 3-5 个子项）
- 右侧放用户头像和通知铃铛图标

### FlowChart（流程图）

- ECharts graph 类型渲染浮选工艺流程
- 节点：磨矿 → 分级 → 粗选 → 扫选 → 精选等环节
- 不同颜色区分铅/锌/尾矿流向
- 当前活跃节点有发光/脉冲动画
- 带箭头曲线连线表示物料流向
- 边框发光效果突出显示

### RealtimeMonitor（实时检测）

- 3-4 个关键指标实时曲线：pH 值、药剂用量、矿浆浓度、泡沫层厚度
- 每个指标用小卡片：当前值 + 迷你折线图
- 数值变化颜色闪烁：正常绿/警告黄/异常红

### ParamTable（参数表格）

- Ant Design 可编辑表格
- 列：参数名、当前值、设定值、单位、状态
- 设定值列可直接点击编辑
- 状态列用颜色标签（正常/偏离/异常）
- 支持分页和搜索

### AlarmPanel（报警区域）

- 顶部报警统计：当前报警数、已处理数、未处理数（徽章样式）
- 报警列表：时间、报警类型、描述、级别
- 级别颜色：紧急（红）/警告（橙）/提示（蓝）
- 支持点击查看详情和标记已处理

### ChartPanel（数据面板+多图表）

- Tab 切换：趋势图、对比图、分布图
- 趋势图：多工艺指标时间序列折线图
- 对比图：铅精矿/锌精矿品位回收率柱状图
- 分布图：关键参数分布直方图
- 时间范围筛选：近1小时/6小时/24小时
