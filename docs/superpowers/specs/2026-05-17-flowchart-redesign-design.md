# 浮选工艺流程图重设计

## 目标

重写 `FlowChart.jsx`，用 ECharts graph + `path://` 自定义 SVG 符号，实现符合工业 P&ID 风格的浮选工艺流程图，包含完整的铅锌浮选连线关系和图例。

## 技术方案

- ECharts graph，`layout: 'none'`，手动坐标定位
- `symbol: 'path://<svg-path>'` 自定义节点图标
- 静态连线（无脉冲动画）
- ECharts `graphic` 组件绘制图例

## 节点定义

## 连线定义

### 实线（尾矿流向）

| 序号 | 起点 | 终点 | 备注 |
|------|------|------|------|
| 1 | 原矿 | 球磨机/旋流器 | 水平 |
| 2 | 球磨机/旋流器 | 铅快粗选 | 水平 |
| 3 | 铅快粗选 | 铅粗选 | 向下 |
| 4 | 铅粗选 | 铅扫选 | 水平 |
| 5 | 铅扫选 | 锌快粗选 | 斜向下 |
| 6 | 锌快粗选 | 锌粗选 | 水平（右→左） |
| 7 | 锌粗选 | 锌扫选 | 水平（右→左） |
| 8 | 锌精选 | 锌快粗选 | 循环回路，curveness: 0.3 |

### 虚线（泡沫流向）

| 序号 | 起点 | 终点 | 备注 |
|------|------|------|------|
| 1 | 铅快粗选 | 铅快精选 | 向上 |
| 2 | 铅快精选 | 铅精选 | 水平 |
| 3 | 铅精选 | 铅精矿 | 水平 |
| 4 | 铅扫选 | 铅粗选 | 回路 |
| 5 | 锌快粗选 | 锌精选 | 水平（右→左） |
| 6 | 锌精选 | 锌精矿 | 水平 |
| 7 | 锌扫选 | 锌粗选 | 回路 |
| 8 | 锌粗选 | 锌快粗选 | 循环回路，curveness: 0.3 |

### 连线样式

- 实线：`lineStyle: { type: 'solid', width: 2, color: '#888' }`
- 虚线：`lineStyle: { type: 'dashed', width: 2, color: '#888' }`
- 箭头：`edgeSymbol: ['none', 'arrow']`，`edgeSymbolSize: [4, 10]`
- 回路连线加 `curveness: 0.3`
